# ez-video-capture

A monorepo for video capture related crates, built on top of FFmpeg via [`rsmpeg`](https://github.com/larksuite/rsmpeg).

## Crates

- **ez-video-capture-core** — Core library for video capturing, decoding, and recording.
- **ez-video-capture-python** — Python bindings for the core library via [PyO3](https://pyo3.rs/).
